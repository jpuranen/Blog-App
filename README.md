# Blog-App
https://github.com/jpuranen/Blog-App

### Compiling and Running the app
1. Install Maven
2. Compile the source code of the project
```
mvn compile
```
3. Run the application using the Maven plugin
```
mvn spring-boot:run
```

### Blog site UI

http://localhost:8080/index.html
