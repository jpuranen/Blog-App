# Blog-App
https://github.com/jpuranen/Blog-App

### Compiling and Running the app
1. Install Node.js
2. Install Maven
```
npm install maven
```
3. Compile the source code of the project
```
mvn compile
```
4. Run the application using the Maven plugin
```
mvn spring-boot:run
```

### Blog site UI

http://localhost:8080/index.html
